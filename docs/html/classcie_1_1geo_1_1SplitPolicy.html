Detailed Description

template<concepts::Iterator PointIterator, concepts::Iterator ValueIterator>
class cie::geo::SplitPolicy< PointIterator, ValueIterator >

Interface for providing points around which to split a cell. Generating the split point is based on a set of point-value pairs and must be implemented in derived classes.
