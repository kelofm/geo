Detailed Description

template<concepts::Primitive PrimitiveType>
class cie::geo::AbsCell< PrimitiveType >

Node interface of a tree of primitives, splittable around a single point. The 'split' interface supplies a set of tuples necessary for constructing a set of primitives (results of the split).
